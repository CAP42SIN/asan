1. github 가입
git다운로드 설치(https://www.git-scm.com/)


2.내 컴퓨터에 git을 설정한 폴더와 파일들을 정한다(만든다).


3. git bash 열기 -> git을 설정한 폴더로 간다.
(폴더 오른쪽 마우스 눌러서 git bash here로 바로 연결 가능)

<명령어>
pwd -> 현재 위치 확인
cd .. -> 상위 디렉토리
cd ~ -> 최상위, 루트로 이동 
ls -> 현재 위치해 있는 디렉토리에 폴더와 파일을 보여줌
cd d: D드라이브로 이동
cd 폴더명 -> 폴더명 폴더로 이동


4. 3에서 정한 폴더를 git init (저장소로 만들기)

ls -al (숨김파일까지 다 보임)
git log (변경 이력 확인)
git status (현재 상태 확인)
